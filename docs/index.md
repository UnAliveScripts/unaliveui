# Welcome

Welcome to the official API documentation for the [UnAliveUI](https://github.com/UnAliveScripts/unaliveui) Library. You can use these docs as reference to build interfaces with the UnAlive API, or use it as a reference for your own UI Library.

## Components

UnAlive includes a growing library of macOS-inspired components:

- [Window](Components/Window.md) — Main window container
- [Card](Components/Card.md) — Content card with shadow
- [Toggle](Components/Toggle.md) — On/off switch
- [Slider](Components/Slider.md) — Range slider
- [Button](Components/Button.md) — Action button
- [TextField](Components/TextField.md) — Text input
- [Notification](Components/Notification.md) — Popup alerts
- And more...
