# Accents

Accents define the primary interactive color used throughout the UI.

## Usage

Accent colors are applied to:

- **Toggle** — active background color
- **Slider** — fill track color
- **Button** — action button background
- **Selection** — highlight and indicator colors

## Accent Colors

| Accent | Hex       | Preview                                             |
| ------ | --------- | --------------------------------------------------- |
| Red    | `#FF4254` | `#!luau Color3.fromRGB(255, 66, 84)` — Default      |
| Blue   | `#0A82FF` | `#!luau Color3.fromRGB(10, 130, 255)`                |
| Green  | `#34C759` | `#!luau Color3.fromRGB(52, 199, 89)`                 |
| Yellow | `#FFBD2E` | `#!luau Color3.fromRGB(255, 189, 46)`                |

## Custom Accents

You can define custom accent colors by passing any `Color3` value to components that support it.
