# Accents

Accent colors control the primary color used for interactive elements throughout the UI.

## Default Accent

The default accent color is `#FF4254` (UnAlive Red).

## Using Accents

Accent colors are used by components including:

- **Toggle** active (on) state background
- **Slider** fill track
- **Button** primary action background
- **Selection** indicators and highlights

## Accent Colors

| Accent | Hex       | Description        |
| ------ | --------- | ------------------ |
| Red    | `#FF4254` | Default accent     |
| Blue   | `#0A82FF` | Standard blue      |
| Green  | `#34C759` | Success state      |
| Yellow | `#FFBD2E` | Warning state      |
