# Interface Guidelines

UnAlive follows macOS-inspired design principles to create clean, intuitive user interfaces.

## Design Philosophy

- **Clarity**: Every element has a clear purpose. Avoid clutter.
- **Depth**: Use shadows, blur, and layering to establish visual hierarchy.
- **Consistency**: Reuse components and spacing patterns throughout your UI.
- **Feedback**: Every interaction should provide immediate visual feedback.

## Spacing

- Use 8px as your base unit for spacing.
- Content padding: 16px
- Section spacing: 8px
- Component gaps: 6px–12px

## Typography

- Use `Gotham` font family for consistency.
- Headings: 16px, SemiBold
- Body text: 14px, Regular
- Captions/Labels: 12px, Regular
- Buttons: 13px, SemiBold

## Colors

UnAlive offers a dark theme by default with the Alert color scheme:

- **Background**: `#0C0C0E`
- **Surface**: `#141418`
- **Border**: `#222228`
- **Text Primary**: `#FFFFFF`
- **Text Secondary**: `#B4B4BE`
- **Accent**: `#FF4254`
